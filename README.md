[Mapping](https://github.com/libla/Mapping)是C#实现的比较简单的轻度orm库，其核心思路为：
1. 所有接口都为异步；
2. 以表达式树为查询条件，以做到编译期类型安全；
3. 尽量抹平不同数据库的差异，统一在同一个对外接口下，并且所有对外接口都有效(因此只会取不同数据库的并集特性，所具有功能比较少)。

* 目前只对MySql做了包装，其他数据库的支持会逐渐加上
